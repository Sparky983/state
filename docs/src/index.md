# Getting Started

## Installation

::: code-group

````xml [Maven]
<dependency>
    <groupId>me.sparky983.state</groupId>
    <artifactId>state</artifactId>
    <version>0.1.0</version>
</dependency>
````

````kt [Gradle (Kotlin DSL)]
repositories {
    mavenCentral()
}

dependencies {
    implementation("me.sparky983.state:state:0.1.0")
}
````

````groovy [Gradle (Groovy DSL)]
repositories {
    mavenCentral()
}

dependencies {
    implementation 'me.sparky983.state:state:0.1.0'
}
````

:::
